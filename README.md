Mana Storm - Multiplayer Game

Complete multiplayer game built with React, Three.js, and Socket.io.

## Quick Start

### Client Setup
```bash
cd client
npm install
npm run dev
```

### Server Setup (in another terminal)
```bash
cd server
npm install
node src/index.js
```

### Access the Game
- Open browser to: http://localhost:3000
- Server runs on: http://localhost:3001

## Features
- Multiplayer with Socket.io
- 5 character classes (Warrior, Mage, Rogue, Archer, Healer)
- 15+ skills with cooldowns
- Party system
- Quest system
- 3D world with Three.js
- First-person WASD + mouse controls
- Real-time chat
